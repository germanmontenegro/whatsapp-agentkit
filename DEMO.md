# 🎬 Guion de Demo — VIP (Lavadero PITA)
### Plataforma: VM — Virtual Machine

> Objetivo: mostrar en vivo cómo el agente atiende, cotiza, agenda turnos y
> retiene clientes. Tiempo estimado: 3-5 minutos.

---

## Antes de empezar (preparación)

1. Abrí la terminal en la carpeta del proyecto:
   ```
   cd C:\Users\PC-USER\Desktop\whatsapp-agentkit
   set PYTHONIOENCODING=utf-8       (en PowerShell: $env:PYTHONIOENCODING="utf-8")
   python tests/test_local.py
   ```
2. Vas a ver el banner de **VM — Virtual Machine**. Ahí arranca la demo.
3. Tip: escribí `limpiar` antes de empezar para arrancar con la charla en limpio.

---

## La secuencia (copiá y pegá estos mensajes, uno por uno)

Hacé una pausa después de cada respuesta para que el cliente la lea. Frases para
acompañar entre paréntesis.

**1.** (Mostrá que entiende el negocio)
```
Hola! que servicios tienen?
```
→ *"Mirá cómo presenta TODO tu negocio al instante, con promos incluidas."*

**2.** (Mostrá que vende)
```
Cuanto sale un lavado completo de camioneta?
```
→ *"Da el precio, ofrece el combo y le calcula el descuento. Está vendiendo."*

**3.** (Mostrá que agenda)
```
Buenisimo. Tenes turno para el sabado a la mañana?
```
→ *"Toma turnos solo, 24 horas, sin que vos contestes."*

**4.** (Mostrá que tiene memoria)
```
Si, sabado 10am. Es una Toyota Hilux
```
→ *"Se acuerda de todo en la conversación, como un empleado atento."*

**5.** (Mostrá el caso 'mal tiempo')
```
Una ultima, si llueve atienden igual?
```
→ *"Hasta maneja los días de lluvia y ofrece reprogramar. No pierde al cliente."*

## Cierre de la demo (lo que decís vos)

- "Esto atiende a tus clientes **las 24 horas**, aunque estés lavando o durmiendo."
- "No perdés **ninguna consulta** por no contestar a tiempo."

- "Lo dejo andando en tu WhatsApp y yo me encargo del mantenimiento."
